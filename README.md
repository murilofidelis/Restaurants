# Meat - Angular App Starter

### Instalando as Dependências

`npm install`

### Inicializando o Servidor

`ng serve` ou `npm start`

### Inicializando o servico

`json-server db.json`

Expressões regulares usadas na validação de formulários

### Email Regex

`/^(([^<>()\[\]\.,;:\s@\"]+(\.[^<>()\[\]\.,;:\s@\"]+)*)|(\".+\"))@(([^<>()[\]\.,;:\s@\"]+\.)+[^<>()[\]\.,;:\s@\"]{2,})$/i`

### Number Regex

`/^[0-9]*$/`
